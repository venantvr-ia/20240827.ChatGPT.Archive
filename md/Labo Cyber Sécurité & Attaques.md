## Introduction au labo de cybersécurité EVE-NG

| Tags |
|------|
| `cybersécurité` `EVE-NG` `hacking éthique` `tests de pénétration` `vulnérabilités` |

Bienvenue dans cet environnement de laboratoire de cybersécurité basé sur EVE-NG. Ce cours s'adresse à tous, débutants ou experts en cybersécurité, hacking éthique, tests de pénétration, tests de vulnérabilité, et chasse aux bugs.

Ce cours vous guidera dans le développement et la configuration d'un laboratoire dans EVE-NG. Nous convertirons des images en format QCOW2 pour les utiliser. Les serveurs vulnérables principaux incluent OWASP, Buggy Web Application, PHP action web app, et Metasploitable.

Nous explorerons l'injection SQL, le cross-site scripting, et plus sur ces serveurs. En plus, nous aurons un serveur web Linux, un serveur web Windows, et des clients pour les tester. Un serveur DHCP et DNS seront également configurés.

Au-delà de la configuration du laboratoire, nous simulerons diverses cyberattaques, notamment les 20 premières attaques, comme la reconnaissance, le flooding MAC, la mise à jour ping, le flooding ICMP, le flooding SYN, Smurf, LAN, Slowloris, l'usurpation ARP, la famine DHCP, l'usurpation DHCP, les attaques par mot de passe, phishing, injection SQL, et DNS.

Nous aborderons les bases de la cybersécurité : introduction, modèle de défense, importance, certifications, architecture Zero Trust, couches du web, meilleurs OS et matériel pour le hacking, top 10 OWASP, et applications web vulnérables. Nous détaillerons l'installation d'EVE-NG et de VMware, ainsi que la conversion d'images QCOW, et le téléchargement de routeurs et commutateurs.

L'objectif principal est de créer ces serveurs vulnérables (Buggy Web Application, Metasploitable, Top 10 OWASP, PHP action web app) et d'installer les principaux systèmes d'attaque (Kali Linux, BackBox, Parrot OS) pour mener les exercices.

Ce cours vous aidera à construire un laboratoire dans EVE-NG et à effectuer des attaques, vous orientant vers le hacking éthique, les tests de pénétration, ou la cybersécurité. Merci.

Les vingt premières attaques incluront des attaques de reconnaissance, saturation MAC, ping, ICMP, paquets ASCII, smurf, LAN, slowloris, ARP spoofing, DHCP starvation, et des attaques par mot de passe et phishing, injection SQL, ainsi que des attaques DNS.

Le cours abordera aussi les bases de la cybersécurité, le modèle de défense, l'importance de la cybersécurité, la certification, l'architecture Zero Trust, les couches du web, les meilleurs systèmes d'exploitation et équipements de hacking, ainsi que l'OWASP Top 10 et les applications web vulnérables.

De plus, il traitera la création de serveurs vulnérables tels que Buggy Web Application, Metasploitable, OWASP Top 10, et PHP Action Web Application. Les quatre principaux systèmes d'attaquant comme Kali Linux, bakebox et Parrot OS seront également installés et utilisés.

Ce cours vous aidera à construire un laboratoire dans EVE-NG et à mener des simulations d'attaques. Il vous accompagnera dans votre parcours, que ce soit dans le domaine du hacking éthique, des tests de pénétration ou de la cybersécurité.

**Dictionnaire de correspondances :**

*   **EVE-NG :** Plateforme de virtualisation réseau open-source pour la création de réseaux virtuels complexes.
*   **QCOW2 :** Format de fichier de disque dur virtuel utilisé par QEMU et KVM.
*   **OWASP :** Open Web Application Security Project - communauté dédiée à la sécurité des applications web.
*   **PHP :** Langage de script côté serveur pour le développement web.
*   **SQL :** Structured Query Language - langage de programmation pour la gestion de bases de données.
*   **DNS :** Domain Name System - système de traduction de noms de domaine en adresses IP.
*   **DHCP :** Dynamic Host Configuration Protocol - protocole d'attribution automatique d'adresses IP.
*   **ICMP :** Internet Control Message Protocol - protocole de contrôle et de messages d'erreur.
*   **ARP :** Address Resolution Protocol - protocole d'association d'adresses IP et MAC.
*   **LAN :** Local Area Network - réseau local.
*   **OS :** Operating System - système d'exploitation.
*   **Kali Linux :** Distribution Linux axée sur la sécurité informatique et le hacking éthique.
*   **Parrot OS :** Distribution Linux basée sur Debian pour la sécurité informatique.
*   **OWASP Top 10 :** Liste des dix principales vulnérabilités de sécurité des applications web.
*   **Hacking éthique :** Tests d'intrusion éthiques pour identifier et corriger les failles de sécurité.
*   **Phishing :** Technique d'ingénierie sociale pour obtenir des informations personnelles.
