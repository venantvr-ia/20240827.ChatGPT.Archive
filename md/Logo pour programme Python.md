## Logo pour programme Python

| Tags |
|------|
| `Python` `Logo` `Graphisme` |

Le but est de créer un logo pour un programme Python.

Voici une proposition utilisant le module `turtle` :

```python
import turtle

# Configuration de l'écran
screen = turtle.Screen()
screen.setup(width=600, height=600)
screen.bgcolor("white")
screen.title("Logo Python")

# Configuration de la tortue
pen = turtle.Turtle()
pen.speed(0)  # Vitesse la plus rapide
pen.hideturtle()
pen.color("blue")

# Fonction pour dessiner le "P" de Python
def draw_p(x, y, size):
    pen.penup()
    pen.goto(x, y)
    pen.pendown()
    pen.setheading(90)  # Orientation vers le haut
    pen.forward(size)
    pen.setheading(0)
    pen.circle(size/2, 180)

# Dessiner le logo
draw_p(-50, 0, 100)

turtle.done()
```

Ce code génère un logo simple avec la lettre "P" stylisée, représentative de "Python".  L'arrière-plan est blanc et la lettre est bleue. La taille et la position peuvent être ajustées en modifiant les paramètres dans le code.

Pour des logos plus complexes, d'autres modules graphiques comme `PIL` (Pillow) ou des bibliothèques de dessin vectoriel comme `SVGwrite` peuvent être utilisées. L'export du logo peut se faire au format PNG, SVG ou autre.
